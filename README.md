# NESAAU Management Portal
**Transparency & Accountability for the 2025/2026 Academic Session**

## 📌 Overview
The NESAAU Portal is a professional administrative tool designed for the Nigeria Economics Students' Association (Adeleke University). It serves as a central hub for financial tracking, executive management, and student announcements, ensuring all departmental operations are handled with precision.

## 🚀 Core Features
* **Membership Registry:** A verified directory of all economics students.
* **Financial Overview:** Live tracking of dues submissions (Paid, Pending, Unpaid).
* **Leadership Council:** Real-time management of current executive profiles.
* **Personnel Spotlight:** Weekly recognition of outstanding members.
* **Automated Archiving:** Seamless transition between academic sessions.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+).
* **Backend/Database:** Supabase (PostgreSQL & Auth).
* **Icons:** Font Awesome 6.
* **Deployment:** Vercel & GitHub.

## 🛡️ Setup & Security
To maintain security, environment variables are managed via a `.env` file. 
**Required Keys:**
- `SUPABASE_URL`
- `SUPABASE_KEY`

*Note: The `.env` file is excluded from version control to protect database integrity.*
